**Smart Assistive Shoe for Visually Impaired:**
A sensor-based wearable system designed to assist visually impaired individuals by detecting obstacles and terrain hazards in real time. The system alerts the user using vibration and buzzer signals.

**Components Used:**
Arduino Microcontroller
Ultrasonic Sensor
IMU Sensor (Accelerometer + Gyroscope)
Buzzer
Vibration Motor
Rechargeable Battery Module
Connecting Wires

**Working:**
Ultrasonic sensor detects obstacles in front of the user.
IMU sensor detects ground conditions such as stairs or uneven surfaces.
Arduino processes sensor data.
When a hazard is detected, the buzzer and vibration motor alert the user.

**Features:**
Real-time obstacle detection
Terrain and stair detection
Dual alert system (vibration + buzzer)
Offline working system
Wearable design
Rechargeable battery powered
